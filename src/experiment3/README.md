# Exp 3 — React Router Demo

This is **Experiment 3**: a small, self-contained demo to satisfy the assignment requirement: create a second page and use React Router to navigate between pages.

Structure:
- `App.jsx` — experiment root (uses `react-router-dom`)
- `components/` — `Navbar`, `Footer`
- `pages/` — `Home`, `Projects`

How to use:
1. Install dependencies (project root): `npm install` (make sure `react-router-dom` is installed).
2. Start the main app: `npm start`.
3. In the app, open the header and click **Experiment 3** (or navigate to the Experiment 3 page from Home) to view this demo.

Notes:
- Router is local to this experiment, so it doesn't change the rest of the app navigation.
- Keep design consistent with other experiments by reusing fonts and spacing.
